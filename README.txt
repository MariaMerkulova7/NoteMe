В данном репозитории представлена реализация веб-приложения онлайн-блокнота "NoteMe".

В репозитории Вы найдёте:
1)код программы main.py;
2)папка db с базой данных;
3)папка data с информацией, необходимой для работы с базой данных;
4)папка static с картинками и стилями;
5)папка templates с шаблонами страниц;
6)папка forms с фласк-формами;
7)файл requirements.txt с библиотеками;
8)пояснительная записка;
9)презентация
